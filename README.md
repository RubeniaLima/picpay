<p>O PicPay Simplificado é uma plataforma de pagamentos simplificada. Nela é possível depositar e realizar transferências de dinheiro entre usuários. Temos 2 tipos de usuários, os comuns e lojistas, ambos têm carteira com dinheiro e realizam transferências entre eles.
</p>



Tecnologias utilizadas

* Java 21
* Spring Boot
* Spring Data JPA
* MySQL
* Docker
* Spring Cloud Open Feign
* ControllerAdvice & Problem Details
* Hibernate Validator
